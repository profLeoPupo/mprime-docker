# 🚀 Mprime Docker - Easy GIMPS Setup

Run mprime (GIMPS) easily using Docker, with minimal setup.

---

## 🎯 Objective

This project simplifies participation in the GIMPS project:

- No manual environment setup
- Works on Linux and Windows
- Persistent data
- Easy scaling to multiple machines

---

## 📦 Requirements

- Docker installed
- Git installed

---

## 🐧 Linux Setup

```bash
git clone https://github.com/SEU_USUARIO/mprime-docker.git
cd mprime-docker/linux
chmod +x setup.sh
./setup.sh
