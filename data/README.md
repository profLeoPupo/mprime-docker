# Runtime Data Directory

This folder stores all runtime data generated by mprime.

## 📦 Contents

- worktodo.txt → assigned work
- results.txt → completed results
- prime.txt → configuration file
- *.bu / *.bu2 → checkpoint files
- logs and temporary data

## ⚠️ Important

- This folder is automatically populated during setup
- Do NOT delete after configuration
- This data ensures progress is not lost

## 🔁 Portability

You can copy this folder to another machine to resume work.
